# OMSI 2 Configuration

## Installation

Just copy+paste the OMSI 2 folder into the parent folder of your OMSI 2 installation.
This is likely `C:\Program Files (x86)\Steam\steamapps\common`.

### Dependencies

ailist.cfg requires:

- Paid: [SP Lite](https://www.studiopolygon.co.uk/litedf)
- Free: [WESTbus](https://fellowsfilm.com/downloads/first-west-of-england-westbus-streetlite.5301/)
- Free: [First Olympia](https://fellowsfilm.com/downloads/ono-first-olympia-repaints.3030/)
- Free: [First Ultraviolet](https://fellowsfilm.com/downloads/first-bus-ultraviolet-streetlite-pack.5211/)
- Free: [V3D Enviro 200 MMC](https://fellowsfilm.com/downloads/adl-enviro-200-mmc.1989/)

## Description

The Westcountry AI List has First running most services with Stagecoach running the P&R.
This is roughly based on how things operate in Bristol where Stagecoach run the Portway P&R.
